# Senna's AI Agent Toolbox

AI agent extensions focused on software development and productivity, with some fun stuff sprinkled on top.

## Installation

```bash
# Claude Code:
claude plugin marketplace add lucasdesenna/ai-agent-toolbox@release
```

```bash
# Codex:
codex plugin marketplace add lucasdesenna/ai-agent-toolbox --ref release
```

## Plugins

### LLM Quests

Quests are structured investigations for everyday software engineering work: technical research, cross-team coordination, feature development, bug fixes, and documentation. A quest starts from a vague trigger such as a Jira card, Slack thread, meeting note, or idea, then moves through lifecycle phases until it reaches a delivered and documented outcome.

The framework is distributed as portable skill packages, and should be agent-agnostic.
